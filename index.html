Etiquetas HTML

Etiqueta Descripción
div
Elemento “contendedor” de otras cosas. Realmente es lo que más se usa. Sabiendo manejar un los estilos, puedes hacer casi todo.
img
Etiqueta con la que se insertan las imágenes, no necesita etiqueta de cierre
ul
Etiqueta de lista no ordenada
a
Añade enlaces. Dentro de ella tendremos dos atributos importantes, que son:
href=”indica_la_URL_o_RUTA” y target=”self o _blank”,
según quieras que el enlace se abra en la misma página o en una pestaña nueva.
p
Determina un párrafo, requiere etiqueta de cierre pero este tiene la ventaja de agregar doble salto de linea para separar lo que siga.
iframe
Inserta contenido de otras páginas
table
Crea tablas, pero debes usar las etiquetas tr para las filas y td para las celdas
b
Aplica negrita a la selección, reemplaza el antiguo "strong"
i
Aplica cursiva a la selección
br
Salto de linea simple

Colors Especificar el color para una etiqueta Con CSS, los colores se pueden especificar de diferentes maneras:
  • Por nombres de colores: color: blue;
  • Como valores RGB: color: rgb(0,0,255) se representa como azul
  • Como valores hexadecimales: #0000FF se muestra como azul
  • Con la palabra clave currentcolor:
    color: blue; /* Blue text color */
    border: 10px solid currentcolor; /* Blue border color */
    • backgrounds Especificar el color de fondo de una etiqueta
    Se puede usar background para definir los valores de una o de todas las propiedades siguientes: background-attachment, color, image, position, repeat.     		Usando la etiqueta:
    background: green;
    Usando bg-image y repeat-style

    background: url("test.jpg") repeat-y;

    Usando box y background-color
    background: border-box red;
    background: no-repeat center/80% url("../img/image.png");
    borders Especificar el estilo de bordes de una etiqueta
    La propiedad border permite definir en una única regla todos los bordes de los elementos seleccionados. Se puede utilizar border para definir el o los valores siguientes: border-width, border-style, border-color

    Su Sintaxis es:
    border: [border-width || border-style || border-color | inherit]     		Ejemplo:
    element border: 1px solid #000; }

    text Son parámetros disponibles para dar estilo al texto
    puedes definir varios aspectos    		 Puedes acompañar al texto de una delgada línea decorativa que puede estar en tres posiciones distintas, como underline (el típico subrayado debajo de la línea), through (en el centro de la línea -tachado-) o overline (encima del texto).
    text-decoration:underline;

    Este efecto que provee de cuatro posibilidades: capitalize que convierte en mayúscula la primera letra de cada palabra del texto, como puedes ver en el ejemplo, donde el texto original no es así. uppercase para convertir todas las letras a mayúsculas y lowercase para lo contrario, es decir, convertir todas las letras a minúsculas.
    El valor por defecto es none que no aplica ningún cambio.
    text-transform:capitalize;

    Alinea el texto según convenga. Con left a la izquierda (por defecto), right a la derecha, center centrado, y justify justificado
    text-align:left;

    Produce que la primera línea del párrafo se escriba adentrada (indentada)
    text-indent:30px;
    fonts La propiedad font permite establecer de una sola vez los valores para todas las propiedades:

    font-style, font-variant, font-weight, font-size, line-height y font-family en una hoja de estilo.     		Sintaxis font: [ font-style || font-variant || font-weight ]? font-size [ / line-height ]? font-family

    Ejemplos:

    Ajusta el tamaño de letra a 12pt y el alto de línea a 14pt. La familia tipográfica es sans-serif
    p { font: 12pt/14pt sans-serif }

    Ajusta el tamaño de letra al 80% del elemento padre o al tamaño por defecto (si no hay elemento padre) y la familia tipográfica a sans-serif
    p { font: 80% sans-serif }

    Ajusta el peso tipográfico a '''bold''' (negrita), el estilo de letra a ''italic'' (cursiva), el tamaño de letra a large (grande) y la familia a serif.
    p { font: bold italic large serif }

    position La propiedad position de CSS especifica cómo un elemento es posicionado en el documento.
    Tambien las propiedades top, right, bottom, y left determinan la ubicación final de los elementos posicionados.     		Damos los valores a una caja

    .box {
    width: 100px;
    height: 100px;
    background: red;
    color: white;
    }

    .outer {
    width: 500px;
    height: 300px;
    overflow: scroll;

    } }
    overflow La propiedad CSS overflow-y define qué se debe mostrar cuando el contenido se desborda de los extremos superior e inferior de un elemento en bloque. overflow-y:hidden — Para esconder el texto fuera de la caja
    #div1 { overflow-y: hidden; margin-bottom: 12px;}

    overflow-y:scroll — Para agregar una barra scroll si se necesita
    #div2 { overflow-y: scroll; margin-bottom: 12px;}

    overflow-y:visible — Para mostrar el texto fuera de la caja si se necesita
    #div3 { overflow-y: visible; margin-bottom: 120px;}

    overflow-y:auto — Este es particular, en la mayoría de los navegadores, equivale a scroll
    #div4 { overflow-y: auto; margin-bottom: 120px;}
    float La propiedad CSS float ubica un elemento al lado izquierdo o derecho de su contenedor, permitiendo a los elementos de texto y en línea aparecer a su costado. El elemento es removido del normal flujo de la página, aunque aún sigue siendo parte del flujo (a diferencia del posicionamiento absoluto).
    Un elemento flotante es un elemento en el que el valor calculado de float no es igual a none.     		Su sintaxis:

    float: left;
    float: right;
    float: none;
    float: inline-start;
    float: inline-end;

    Valores globales
    float: inherit;
    float: initial;
    float: unset;

    Ejemplo:

    section {
    border: 1px solid blue;
    width: 100%;
    float: left;
    }

    div {
    margin: 5px;
    width: 50px;
    height: 150px;
    }

    .left {
    float: left;
    background: pink;
    }

    .right {
    float: right;
    background: cyan;
    }
