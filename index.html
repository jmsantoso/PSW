Etiquetas HTML

Etiqueta Descripción
div
Elemento “contendedor” de otras cosas. Realmente es lo que más se usa. Sabiendo manejar un los estilos, puedes hacer casi todo.
img
Etiqueta con la que se insertan las imágenes, no necesita etiqueta de cierre
ul
Etiqueta de lista no ordenada
a
Añade enlaces. Dentro de ella tendremos dos atributos importantes, que son:
href=”indica_la_URL_o_RUTA” y target=”self o _blank”,
según quieras que el enlace se abra en la misma página o en una pestaña nueva.
p
Determina un párrafo, requiere etiqueta de cierre pero este tiene la ventaja de agregar doble salto de linea para separar lo que siga.
iframe
Inserta contenido de otras páginas
table
Crea tablas, pero debes usar las etiquetas tr para las filas y td para las celdas
b
Aplica negrita a la selección, reemplaza el antiguo "strong"
i
Aplica cursiva a la selección
br
Salto de linea simple

Colors Especificar el color para una etiqueta Con CSS, los colores se pueden especificar de diferentes maneras:
  • Por nombres de colores: color: blue;
  • Como valores RGB: color: rgb(0,0,255) se representa como azul
  • Como valores hexadecimales: #0000FF se muestra como azul
  • Con la palabra clave currentcolor:
    color: blue; /* Blue text color */
    border: 10px solid currentcolor; /* Blue border color */

    height La propiedad CSS height especifica la altura de un elemento. Por defecto, la propiedad define la altura del área de contenido. Ejemplos: div { width: 250px; margin-bottom: 5px; border: 2px solid blue; } #taller { height: 50px; } #shorter { height: 25px; } #parent { height: 100px; } #child { height: 50%; width: 75%; }

    tables Permite dar un formato a las tablas, cambiar el espaciado y la distribución Lo primero que hay que hacer es solucionar los aspectos de espaciado/distribución; que por defecto en la tabla son muy justos, la tipografía, Gráficos y colores Darle un estilo al titulo que destaque /* spacing */ table { table-layout: fixed; width: 100%; border-collapse: collapse; border: 3px solid purple; } thead th:nth-child(1) { width: 30%; } thead th:nth-child(2) { width: 20%; } thead th:nth-child(3) { width: 15%; } thead th:nth-child(4) { width: 35%; } th, td { padding: 20px; } /* typography */ html { font-family: 'helvetica neue', helvetica, arial, sans-serif; } thead th, tfoot th { font-family: 'Rock Salt', cursive; } th { letter-spacing: 2px; } td { letter-spacing: 1px; } tbody td { text-align: center; } tfoot th { text-align: right; } thead, tfoot { background: url(leopardskin.jpg); color: white; text-shadow: 1px 1px 1px black; } thead th, tfoot th, tfoot td { background: linear-gradient(to bottom, rgba(0,0,0,0.1), rgba(0,0,0,0.5)); border: 3px solid purple; } caption { font-family: 'Rock Salt', cursive; padding: 20px; font-style: italic; caption-side: bottom; color: #666; text-align: right; letter-spacing: 1px; }
    position La propiedad position de CSS especifica cómo un elemento es posicionado en el documento. Tambien las propiedades top, right, bottom, y left determinan la ubicación final de los elementos posicionados. Damos los valores a una caja .box { width: 100px; height: 100px; background: red; color: white; } .outer { width: 500px; height: 300px; overflow: scroll; padding-left: 150px; } }
    overflow La propiedad CSS overflow-y define qué se debe mostrar cuando el contenido se desborda de los extremos superior e inferior de un elemento en bloque. overflow-y:hidden — Para esconder el texto fuera de la caja #div1 { overflow-y: hidden; margin-bottom: 12px;} overflow-y:scroll — Para agregar una barra scroll si se necesita #div2 { overflow-y: scroll; margin-bottom: 12px;} overflow-y:visible — Para mostrar el texto fuera de la caja si se necesita #div3 { overflow-y: visible; margin-bottom: 120px;} overflow-y:auto — Este es particular, en la mayoría de los navegadores, equivale a scroll #div4 { overflow-y: auto; margin-bottom: 120px;}
    float La propiedad CSS float ubica un elemento al lado izquierdo o derecho de su contenedor, permitiendo a los elementos de texto y en línea aparecer a su costado. El elemento es removido del normal flujo de la página, aunque aún sigue siendo parte del flujo (a diferencia del posicionamiento absoluto). Un elemento flotante es un elemento en el que el valor calculado de float no es igual a none. Su sintaxis: Sintáxis /* Valores clave || Keyword values */ float: left; float: right; float: none; float: inline-start; float: inline-end; /* Valores globales || Global values */ float: inherit; float: initial; float: unset; Ejemplo: section { border: 1px solid blue; width: 100%; float: left; } div { margin: 5px; width: 50px; height: 150px; } .left { float: left; background: pink; } .right { float: right; background: cyan; }
